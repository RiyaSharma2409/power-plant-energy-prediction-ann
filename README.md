# ⚡ Power Plant Energy Prediction using ANN

## 🚀 Overview
This project predicts energy output of a power plant using an Artificial Neural Network.

---

## 📊 Dataset
- Combined Cycle Power Plant dataset  
- Features: Temperature, Pressure, Humidity, etc.  
- Target: Energy Output

---

## ⚙️ Approach
- Train-test split  
- Built ANN with:
  - 2 hidden layers (ReLU)
  - Output layer (Linear)

---

## 🧠 Model Architecture
- Dense (6 units, ReLU)
- Dense (6 units, ReLU)
- Dense (1 unit)

---

## 📈 Results
- Model learns continuous relationships  
- Optimized using Mean Squared Error  

---

## 🧠 Insights
- ANN works well for regression problems  
- Linear output layer used for continuous prediction  
- Captures complex feature interactions  

---

## 🏆 Conclusion
ANN effectively models non-linear relationships in energy prediction.
